% Regeln

% Subfunktion 1: Aktive Studierende (mehr als 2 belegte Kurse)
activeStudent(?S) :- enrolled(?S, ?C1), enrolled(?S, ?C2), enrolled(?S, ?C3), ?C1 != ?C2, ?C1 != ?C3, ?C2 != ?C3.

% Subfunktion 2: Kurs hat gute Bewertung bekommen von aktivem Studenten
goodCourseRatingFromActiveStudent(?C) :- gaveRating(?S, ?C, good), activeStudent(?S).

% Subfunktion 3: Professor hat Kurs mit guter Bewertung unterrichtet
wellRatedProfessor(?P) :- teaches(?P, ?C), goodCourseRatingFromActiveStudent(?C).

% Hauptabfrage: Finde Professoren mit gut bewerteten Kursen von aktiven Studierenden
@output wellRatedProfessor.
